🚀 SCRIPT MAESTRO PARA EJECUTAR TODA LA APLICACIÓN LOCAL

NUEVO: ¡Ahora puedes ejecutar BACKEND + FRONTEND con UN SOLO COMANDO!

1. Abrir una terminal en la raíz del proyecto

2. Ejecutar el script maestro:

./start-dev.sh

3. ¡Eso es todo! Se iniciará automáticamente:
   - Backend en http://localhost:7071/api
   - Frontend en http://localhost:3000

4. Para detener todo:

./stop-dev.sh

5. Para ver el estado:

./status-dev.sh

═══════════════════════════════════════════════════════════════

MANUAL (si prefieres ejecutar por separado):

BACKEND:
cd backend
func start --cors "http://localhost:3000"

FRONTEND (en otra terminal):
cd frontend
npm run dev

═══════════════════════════════════════════════════════════════

SOLUCIÓN DE PROBLEMAS:

Si hay puertos ocupados:
./stop-dev.sh

Si hay errores:
./status-dev.sh

Ver logs en tiempo real:
tail -f logs/backend.log
tail -f logs/frontend.log
